## Getting Started

### Prerequisites

- Ruby (version 3.2.1)
- Rails (version 7.0.4)
- Database PostgreSQL

### Installation

1. Clone the repository:

   ```bash
   git clone https://github.com/AlinaSapronova/exercise_for_kids.git)https://github.com/AlinaSapronova/exercise_for_kids.git```
   
3. ```bundle install```
4. ```rails db:create```
5. ```rails s```
